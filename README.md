## LaraBye（楽曲情報投稿掲示板）

## 概要
好きなアーティストや楽曲のことを投稿でき、ほかの人の投稿できになった楽曲などを調べられる掲示板を作りました。

## 使い方
ユーザー一覧から他ユーザーのフォローやその人のページに飛び投稿の閲覧が可能。

フォローしているユーザーと自分の投稿がタイムライン上に表示されます。

新規ツイートボタンから新規投稿をすることができ、楽曲検索ボタンで楽曲の検索・サンプルが聞けるページに遷移できます。
